# OverWork
A framework for services and controllers, heavily inspired by [Knit](https://github.com/Sleitnick/Knit).

All that's *necessary* is a folder in ServerScriptService named 'Services' and a folder in ReplicatedStorage named 'Controllers'.
Once everything is set up, you simply require the module once from the server and once from the client, then you're good to go.
